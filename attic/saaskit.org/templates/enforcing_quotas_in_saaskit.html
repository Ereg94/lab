Part 7 : Enforcing Quotas to your Polls

Before we start developing the customized version of the Polls app, we would first enforce the necessary quotas as we will be needing the settings for the code. Now, you would like to give the most benefit to your highest paying customer. To enforce such a scheme, it all starts with editing the settings file for your App.

In this sample Polls App we will enforce that the Gold account holders get the most benefit. ie A Gold account holder can create 20 Polls in his/her app, while a silver account will be able to create 10 Polls only.

To do so, open settings.py in a text editor and locate a Setting named - QUOTAS. Here you would see a setting which implements quota for muaccounts django application. To apply quota for the Polls App enter the following string " 'polls': (10,20), " in a new line below muaccounts(without the double quotes). Lets break up the statement to understand it better. Here 'polls' stands for the name of the quota setting which can be anything you prefer. Next is a string of numbers that define the quota limits you want to enforce.

By updating this setting, you will have to re-sync the database in order to apply the same. By re-syncing the database you will create the necessary permissions that needs to be applied to the respective user groups. You can re-sync the database by running the following command from console from the /projects/main/ directory of SaaS Kit: ../projects/main$ python manage.py syncdb

After you have run the command, login to the Admin configuration of the Main site and navigate to Auth>Groups. Here you should see the list of Groups that are present in the database. Click on 'Gold Member' to apply quota settings for Gold Subscriptions. From the Available permissions, select quotas | quotas | 20 polls and add them to Chosen. Then click save and thats it. You have successfully applied a limiting setting of 20 Polls for your Gold Account Subscribers. Similarly, apply a permission of 10 Polls to Silver member group to apply a limit of 10 Polls to you Silver Account Subscribers.


When you're comfortable with this section, read part 8 of this tutorial to get "Integrating the polls app ".
