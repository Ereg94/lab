First things first, Lets begin with an Introduction to SaaS Kit

What is SaaS Kit?

SaaS Kit is a complete SaaS-type website template, ready for deployment. It includes a set of integrated Django applications, ready to change sample templates, together with a Yahoo YUI-based CSS theme. Its dependent on Python 2.5 or higher. Currently the SaaS Kit will work without the need of a database server as it runs on Sqlite which is already installed/available with Python libraries. It is also dependent upon Django framework v1.0.2 or higher. Please read the README for a complete list of dependencies needed for SaaS Kit to work.
